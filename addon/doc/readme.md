# Telegram Justi

Complemento de accesibilidad para Telegram Desktop y Unigram.

Optimiza la navegación mediante atajos de teclado diseñados para usuarios de lectores de pantalla.

## Atajos disponibles

| Atajo | Función |
|---|---|
| Control+R | Grabar o enviar mensaje de voz |
| Barra espaciadora | Reproducir o pausar audio |
| Control+Shift+L | Iniciar llamada |
| Control+Shift+V | Iniciar videollamada |
| Control+Shift+N | Finalizar llamada |
| Control+P | Abrir perfil del contacto o grupo |
| Control+Shift+A | Adjuntar multimedia |
| Control+N | Crear nuevo chat |
| Alt+E | Ir al cuadro de mensaje |
| Control+A | Abrir menú de navegación |

## Funciones

- Grabación y envío de mensajes de voz
- Control de reproducción de audios
- Llamadas y videollamadas
- Apertura rápida de perfiles
- Adjuntar archivos multimedia
- Creación rápida de chats
- Navegación optimizada
- Enfoque automático en lista de conversaciones

## Compatibilidad

- NVDA 2026.1
- NVDA 2026.1.1
