# Changelog

Todos los cambios importantes de Telegram Justi serán documentados en este archivo.

---

## Versión 1.0.0

Fecha de lanzamiento: 2026

### Primera versión pública

#### Funciones añadidas

- Grabación de mensajes de voz.
- Envío rápido de mensajes de voz.
- Reproducción y pausa de mensajes de voz.
- Apertura rápida del perfil del chat o grupo.
- Inicio de llamadas de voz.
- Inicio de videollamadas.
- Finalización de llamadas y videollamadas.
- Acceso rápido a adjuntar multimedia.
- Apertura de nuevo chat.
- Enfoque rápido del cuadro de edición de mensajes.
- Apertura del menú de navegación.
- Enfoque automático en la lista de chats al iniciar Telegram Unigram.

#### Atajos de teclado

| Atajo | Acción |
|---|---|
| Ctrl+R | Grabar mensaje de voz |
| Doble Ctrl+R | Enviar mensaje de voz |
| Espacio | Reproducir / pausar audio |
| Ctrl+Shift+L | Iniciar llamada de voz |
| Ctrl+Shift+V | Iniciar videollamada |
| Ctrl+Shift+N | Finalizar llamada |
| Ctrl+P | Abrir perfil |
| Ctrl+Shift+A | Adjuntar multimedia |
| Ctrl+N | Nuevo chat |
| Alt+E | Enfocar cuadro mensaje |
| Ctrl+A | Abrir menú de navegación |

#### Mejoras

- Optimización de navegación mediante teclado.
- Mejor integración con NVDA.
- Compatibilidad mejorada con Telegram Unigram.
- Mejora de accesibilidad para usuarios ciegos y de baja visión.

#### Correcciones

- Corrección en activación de botones ocultos.
- Corrección en llamadas y videollamadas.
- Corrección en reproducción de mensajes de voz.
- Corrección de enfoque automático en lista de chats.
